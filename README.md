A/B Testing Analysis – Landing Page Conversion Optimization

Analysed an A/B test to evaluate whether a redesigned landing page improved user conversion rates compared to the existing version.

The analysis included data validation, experiment quality checks, hypothesis testing, and interpretation of business impact.

Problem Statement

An e-commerce company launched a new landing page design aimed at increasing conversions.

The objective was to determine whether the new design produced a statistically significant improvement over the existing page.

Analysis Performed
Cleaned and validated experiment data
Removed duplicate users and invalid page assignments
Checked traffic split consistency between groups
Compared conversion performance between control and treatment groups
Performed one-tailed two-proportion z-test
Calculated confidence intervals and uplift metrics
Conducted power analysis and sample size evaluation
Results
Metric	Value
Control Conversion Rate	11.98%
Treatment Conversion Rate	12.42%
Absolute Lift	+0.44 percentage points
Relative Improvement	+3.63%
P-value	0.0002
Statistically Significant	Yes

The treatment group achieved a statistically significant increase in conversion rate compared to the control group.

While the uplift was relatively small in absolute terms, the impact could still be meaningful at high traffic volumes.

Key Takeaways
Validated experiment integrity before analysis
Evaluated both statistical and practical significance
Demonstrated how small conversion uplifts can scale into larger business impact
Highlighted the importance of sample size and experiment power in A/B testing
